# 新聞稿 email 模板

新聞稿格式高度規律，可拆成「固定字」與「每次填的空格」。實務上由
`scripts/render_press_release.py` 套出，本檔說明格式邏輯，供理解與微調。

## 變數（每次發稿提供）

| 變數 | 範例 | 說明 |
|------|------|------|
| `date` | 2026/05/20 | 發布日期，週幾自動算成 (三) |
| `game` | 遊戲名稱 | 遊戲名稱，會包成《...》 |
| `version` | S00 | 版號，可留空 |
| `update_name` | 改版名稱 | 改版/活動名稱，會包成「...」，可留空 |
| `action` | 今日開服 | 動作，後接「！」 |
| `slogan` | 主標語○○○ | 主標語，後接「！」 |
| `drive_link` | https://drive.google.com/... | 附件連結 |
| `recipients` | ["a@x.com", ...] | 收件 email 陣列，未給則用 config |

## 組裝邏輯

**標題行（同時是 email 主旨）：**
```
[日期(週幾)] [標題外框]《[遊戲]》[版號]「[改版名]」[動作]！[主標語]！
```

**內文：**
```
親愛的媒體朋友您好
 [日期(週幾)] [標題外框]
《[遊戲]》[版號]「[改版名]」[動作]！[主標語]！

附件檔案位置
[Drive 連結]
[固定結尾語]
```
（內文第二行的日期前保留一個半形空格，與來源範例一致。）

## 完整範例

變數：
```json
{
  "date": "2026/05/20",
  "game": "遊戲名稱",
  "version": "S00",
  "update_name": "改版名稱",
  "action": "今日開服",
  "slogan": "主標語○○○",
  "drive_link": "https://drive.google.com/drive/folders/...",
  "recipients": ["media1@example.com", "media2@example.com"]
}
```

產出：
```
主旨：
2026/05/20(三) 【○○新聞稿】《遊戲名稱》S00「改版名稱」今日開服！主標語○○○！

內文：
親愛的媒體朋友您好
 2026/05/20(三) 【○○新聞稿】
《遊戲名稱》S00「改版名稱」今日開服！主標語○○○！

附件檔案位置
https://drive.google.com/drive/folders/...
敬請您協助發稿，非常感謝。

收件人：
media1@example.com
media2@example.com
```

## 微調

- 動作或標語留空時，對應的「！」會自動省略。
- 版號或改版名留空時，該段自動略過，不會留下空的《》或「」。
- 要改外框、結尾語，改 config.json 即可；要單次覆蓋，在本次變數 JSON 放 `title_frame` / `closing`。

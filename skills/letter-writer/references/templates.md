## 1. 詢價/報價信

**適用情境**：向廠商詢問產品/服務價格，或回覆對方的詢價提供報價。

**結構**：稱謂 → 說明來意與需求 → 列出規格/數量/時程 → 請求報價與期限 → 署名

**範例（詢價）**
```
王經理 您好：

我是[公司名稱][部門][姓名]。因業務需求，我們近期計畫採購下列項目，想向貴公司洽詢報價：

1. 品項：[產品名稱／型號]
2. 數量：[數量]
3. 期望交期：[日期]
4. 其他需求：[保固／售後／規格說明]

煩請於[日期]前提供報價單，並註明單價、稅務與付款條件。如需進一步資料，歡迎與我聯繫。

謝謝，期待您的回覆。

[姓名]
[職稱]｜[公司名稱]
[電話]｜[Email]
```

## 2. 合作提案信

**適用情境**：主動向潛在夥伴/客戶提出合作或介紹方案。

**結構**：稱謂 → 自我介紹 → 提出合作構想與對對方的價值 → 具體建議或下一步 → 邀約洽談 → 署名

**範例**
```
[稱謂] 您好：

我是[公司名稱]的[姓名／職稱]。[公司一句話簡介]。

關注到[對方公司／近期動態]，我們認為雙方在[合作領域]上有很好的合作空間。具體而言，我們可以提供[你能帶給對方的價值／資源]，協助貴公司[達成的目標]。

若您有興趣，我很樂意安排一次簡短的線上會議，進一步說明合作細節。不知[時間選項]是否方便？

期待有機會與您交流。

[姓名]
[職稱]｜[公司名稱]
[電話]｜[Email]
```

## 3. 跟進/催覆信

**適用情境**：禮貌追蹤先前已提出、但對方尚未回覆的事項。重點是不失禮又能推進。

**結構**：稱謂 → 承接前次溝通 → 重申需求與重要性 → 明確期望時限 → 體諒語氣 → 署名

**範例**
```
[稱謂] 您好：

先前（[日期]）曾就[事項]與您聯繫，想再與您確認目前的進度。

由於[事項]關係到[原因／後續安排]，希望能在[期限]前收到您的回覆，以利我們安排後續作業。

了解您事務繁忙，若需要我補充任何資料，請隨時告知。感謝您的協助。

[姓名]
[職稱]｜[公司名稱]
[電話]｜[Email]
```

## 4. 道歉信

**適用情境**：對延誤、疏失、品質或服務問題致歉。關鍵是誠懇、承認問題、提出補救，不找藉口。

**結構**：稱謂 → 誠懇致歉 → 簡述狀況（不推諉）→ 具體補救措施 → 預防說明 → 再次致歉 → 署名

**範例**
```
[稱謂] 您好：

針對[問題事件]造成您的不便，我們深感抱歉。

關於[簡述發生的狀況與原因]，責任在我方，未能達到應有的標準。為此，我們將[具體補救措施，例如重新出貨／退款／補償]，並預計於[時間]前完成。

我們已[預防措施]，以確保類似情形不再發生。再次為此次疏失向您致歉，感謝您的諒解與包容。

[姓名]
[職稱]｜[公司名稱]
[電話]｜[Email]
```

## 5. 感謝信

**適用情境**：對協助、合作、款待或機會表達謝意。具體點出對方做了什麼，比泛泛的「謝謝」更有誠意。

**結構**：稱謂 → 表達感謝 → 具體說明對方的貢獻與影響 → 展望未來 → 署名

**範例**
```
[稱謂] 您好：

非常感謝您[對方做的具體事情]。

因為您的[協助／支持／建議]，我們得以[達成的成果或受到的幫助]，這對[團隊／專案]意義重大。

期待未來能有更多合作的機會，也歡迎隨時與我們聯繫。再次致上誠摯的謝意。

[姓名]
[職稱]｜[公司名稱]
[電話]｜[Email]
```

## 6. 通知/公告信

**適用情境**：告知異動、政策更新、時程安排、活動等需周知的事項。重點是資訊清楚、行動明確。

**結構**：稱謂 → 一句話點明主旨 → 條列關鍵資訊（時間/地點/影響）→ 需配合的行動 → 聯絡窗口 → 署名

**範例**
```
[稱謂／敬啟者] 您好：

謹此通知[事項主旨]，相關資訊如下：

- 事項：[說明]
- 生效／實施日期：[日期]
- 影響範圍：[對收件人的影響]
- 需配合事項：[若有]

如有任何疑問，請洽[聯絡窗口／Email／電話]。感謝您的配合。

[姓名／單位]
[公司名稱]
[聯絡方式]
```
